## 🚫 创作约束声明

### 约束 1：成长曲线

```
类型：数值驱动型（线性成长）
  □ 每级+固定属性
  □ 靠等级压制
  □ 装备品质决定战力上限
  □ 经验值积累→升级→变强

防疲劳策略：
  □ 加入副本外的收获维度（图纸/材料/特殊道具）
  □ 每20章换一种副本玩法
  □ 加入基地/安全屋建设线作为副循环
```

### 约束 2：数值膨胀控制

```
  □ 每5级设置一个瓶颈，需要特殊条件突破
  □ 属性增长为递减曲线（初期+5，后期+1）
  □ 每个大阶段设属性上限
  □ 达到上限后只能靠机制/技巧突破
```

### 约束 3：同质化防控

```
阶段玩法多样化：建议
  □ 1-20章：生存适应期
  □ 21-50章：副本发育期
  □ 51-80章：势力冲突期
  □ 81-120章：世界观揭秘期
```
