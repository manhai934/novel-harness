## 五、一句话题材定位

用一句话把你的书说清楚：

```text
全民被投放到什么世界，
主角靠什么优势活下来，
他的基地会成长成什么，
最大的外压是什么。
```

示例：

```text
全民被投放进极寒迷雾世界，每人一间快散架的小木屋。
主角能看到物品备注，一边在雾中找资源，一边把木屋养成能扛住灾潮的庇护所。
```

## 六、核心爽点设计

你的项目至少要明确这五类爽点：

```text
1. 生存爽点：今天又活下来了
2. 升级爽点：基地比昨天更强了
3. 信息差爽点：别人不知道，我知道
4. 对比爽点：别人挨冻挨饿，我已经稳定发育
5. 长线爽点：这个小据点会变成什么
```

建议从里选 2-3 个做主爽点，不要每种都平均发力。

## 七、世界观结构

```text
开局资源紧缺
→ 面对环境/天灾/怪物/黑夜压力
→ 出门探索或交换资源
→ 获得食物/材料/图纸/装备/情报
→ 回到庇护所升级与防守
→ 迎接下一轮更强压力
```

读者追更时主要会问这五件事：

1. 主角这次缺什么。
2. 主角准备去哪里拿。
3. 主角拿到了什么。
4. 主角的木屋/庇护所发生了什么升级变化。
5. 下一轮危机比上一轮强在哪里。

## 八、黄金三章模板

### 第 1 章：先立“今晚怎么活”

必须解决三件事：

- 世界规则突变。
- 主角当前缺口明确。
- 今晚不解决会死或会惨。

### 第 2 章：第一次出门，第一次见到差距

必须解决三件事：

- 第一次探索或尝试。
- 第一次真正收获。
- 第一次和别人形成对比。

### 第 3 章：第一次升级，第一次闭环

必须解决三件事：

- 收获转化成升级。
- 升级立即改变处境。
- 章尾抛出下一轮更大危机。

## 九、子路线速记

### 1. 木屋/庇护所求生

适合新人保底，最稳。

```text
破木屋
→ 一级木屋
→ 带壁炉和仓库的小庇护所
→ 防御型安全屋
→ 可容纳他人的据点
```

核心爽点：

- 从破烂到安全的可视化升级。
- 每次补齐一个生存短板。
- 别人还在挨冻挨饿，主角已经有火、有水、有防线。

### 2. 迷雾求生

适合加探索感和未知感。

核心爽点：

- 雾区未知。
- 视野受限。
- 风险和收益并存。
- 备注/提示类金手指更容易发挥作用。

适合作为 `全民求生` 的世界氛围层，不建议单独悬空存在，最好和 `木屋/庇护所成长` 绑定。

### 3. 公路求生

更偏移动型基地。

核心爽点：

- 站点选择。
- 车体升级。
- 路线风险。
- 流动式资源补给。

### 4. 海岛/海上求生

更偏区域探索和物资点争夺。

核心爽点：

- 岛屿差异。
- 海域风险。
- 漂流与靠岸节奏。

### 5. 领地化生存

这是全民求生的中期升级路线，不是另一个大类。

```text
个人木屋
→ 小院/营地
→ 可保护少量同伴的庇护所
→ 小型安全区
→ 半公共领地
```

核心爽点：

- 主角成果开始被别人需要。
- 资源压力升级为秩序压力。
- “我能活”升级为“我能护住一块地方”。

写法边界：

- 前期不要直接写成全民领主争霸。
- 先有个人生存成绩，再自然吸附他人。
- 先解决火、水、食物、防守，再谈管理、规则、归属。

## 十、新人最稳组合

如果你是新人作者，默认推荐这一条：

`全民求生 + 木屋/庇护所 + 迷雾探索 + 天灾轮换 + 信息提示 + 交易频道`

原因：

- 容易写清楚。
- 容易形成每章反馈。
- 容易做出和别人对比的爽点。
- 金手指不容易失控。
- 中后期能自然升级到领地化生存。

## 十一、前 30 章保底推进模板

```text
1-5章：
立规则、立环境、立木屋、立金手指边界。

6-10章：
第一次稳定获取水/食物/燃料，第一次扛过灾害。

11-15章：
第一次图纸升级，第一次交易占优，第一次拉开与普通人的差距。

16-20章：
第一次深入探索，第一次接触特殊地点或稀有材料。

21-25章：
第一次强化防御或功能房间，第一次让别人意识到主角的庇护所价值。

26-30章：
第一次埋下“领地化生存”钩子，例如收留、合作、防线、区域编号、跨区信息。
```

前 30 章的重点不是解释世界，而是建立追读循环。

前 30 章不要做的事：

- 不大谈世界真相。
- 不直接写大势力争霸。
- 不引入太复杂的文明史。
- 不把金手指写成万能外挂。

## 十二、长期故事发展

全民求生能不能写长，关键看“故事问题”能不能持续升级。

```text
发现缺口
→ 出门探索/交易/挑战
→ 拿到资源/图纸/装备/信息
→ 回屋升级/改造/防守
→ 在频道或现实结果中形成反馈
→ 新灾害/新区块/新矛盾出现
→ 进入下一轮
```

推荐升级顺序：

```text
个人活命
→ 活得稳定
→ 活得舒服
→ 活得比别人强
→ 成为别人依附的对象
→ 卷入区域秩序竞争
```

这条顺序适合新人，因为每一步都能转化成新目标和新爽点。

## 十三、阶段世界观展开

```text
前期：
只解释眼前规则，不解释终极真相。

中期：
用特殊区域、区域编号、异常遗迹轻埋世界更大一层。

后期：
再解释为什么会有全民投放、天灾轮换、区域合并。
```

世界观的任务不是炫设定，而是不断给故事下一阶段找更大的问题。

## 十五、项目自检

```text
这本书的一句话卖点是否清楚？
黄金三章是否能连续形成“缺口-收获-升级”？
主爽点有没有明确优先级？
世界观是否按阶段展开，而不是开局灌设定？
前30章是否已经埋下中期升级的钩子？
```

## 十六、允许 vs 禁止清单

| 行为 | 判定 |
|:-----|:------|
| 资源紧缺、环境压迫、庇护所升级 | ✅ 必须保留 |
| 金手指只给提示和信息差 | ✅ 鼓励 |
| 通过交易频道做对比和打脸 | ✅ 鼓励 |
| 从个人木屋自然升级到小型安全区 | ✅ 鼓励 |
| 开局直接领主争霸 | ❌ 禁止 |
| 开局直接无敌基地 | ❌ 禁止 |
| 金手指无限返还/无限资源/万能商城 | ❌ 禁止 |
| 章节里只有气氛，没有收获反馈 | ❌ 禁止 |
| 迷雾、木屋、领地三者互相割裂 | ❌ 禁止 |
