# 全民求生子题材边界：迷雾木屋求生

> 本文只负责判断这种写法是否跑偏，不负责设计黄金三章和世界观方案。
> 总规则见 `全民求生题材模板.md`。

## 一、核心边界

迷雾木屋求生至少要同时具备：

- `木屋/庇护所` 作为成长核心。
- `迷雾` 作为外部探索和风险空间。
- `资源缺口` 持续存在。
- `探索 -> 带回资源 -> 木屋升级` 的稳定闭环。

如果缺少其中两项以上，基本就不是这个子题材。

## 二、必须保留

- 雾区未知感和风险感。
- 木屋从破烂到安全的成长变化。
- 每次出门都伴随真实代价。
- 资源和环境直接影响当晚生存。
- 升级后生活状态明显改善。

## 三、禁止写法

- 把迷雾写成纯背景滤镜，不参与风险和探索。
- 把木屋写成纯属性面板，不改变生活状态。
- 开局几章内就把故事写成势力争霸。
- 提示系统直接给完整攻略。
- 主角长期不出门，只在屋里囤货升级。

## 四、常见毒点

- 只有开箱，没有场景化探索。
- 雾里永远只有树和箱子，资源点过于单调。
- 每章都在找木头和水，内容重复。
- 木屋升级只有“一级二级三级”，没有功能差异。
- 所有人都在频道里无脑震惊，缺少真实比较逻辑。

## 五、金手指边界

适合：

- 物品备注。
- 危险提示。
- 短期雾潮/天气预警。

不适合：

- 地图全开。
- 无限视野。
- 一键锁定资源点。
- 无冷却的全知提示。

## 六、中期升级红线

- 中期必须引入更复杂的雾中地点，不然会变流水账。
- 中期必须让木屋承担更多功能，不然升级缺乏意义。
- 中期可以走向领地化生存，但不能突然跳成全民领主。

## 七、审稿自检

```text
迷雾有没有真正制造风险？
木屋升级有没有改变生活状态？
资源获取是否仍然有探索成本？
故事有没有从个人求生突然跳成势力争霸？
是否出现了万能提示、全知外挂？
```
