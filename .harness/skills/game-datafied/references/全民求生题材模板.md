# 全民求生题材模板

> 重参考文件。仅在用户选择全民求生、迷雾求生、庇护所升级、小木屋求生等方向，需要搭建保下限的题材结构时加载。

## 一、核心定位

全民求生不是单纯“开箱、升级、打脸”，而是一套可以长期循环的题材模板：

```text
个人生存
→ 庇护所建设
→ 特殊区域探索
→ 区域竞争
→ 安全区经营
→ 世界规则揭示
```

前期靠熟悉模板降低写作难度，中后期靠问题层级升级避免重复疲劳。

## 二、保下限基础模板

适合新人、老书虫转作者、脑洞和智斗不稳定时使用：

```text
全民投放
+ 初始庇护所
+ 基础资源缺口
+ 迷雾/地图探索
+ 信息提示/物品备注
+ 交易频道对比
+ 周期性天灾
+ 庇护所升级
```

默认章节循环：

```text
发现缺口/新目标
→ 探索或挑战
→ 获得资源/图纸/装备/能力
→ 当场形成反馈（升级、交易、打脸、解决危机）
→ 引出新地图/新材料/新灾害/新矛盾
→ 进入下一轮循环
```

## 三、阶段模板

### 第一阶段：个人求生（1-30章）

功能：建立题材、主角能力和基础爽点。

常用内容：

- 开局木屋/草棚/小院。
- 找水、找食物、砍树、取火。
- 第一次开箱、第一次交易、第一次升级。
- 第一次天灾或夜晚怪物袭击。

写作边界：

- 不展开大势力。
- 不解释世界真相。
- 不写复杂文明史。
- 每章给明确反馈。

### 第二阶段：庇护所成型（31-80章）

功能：让读者追“主角的家会变成什么样”。

常用内容：

- 壁炉、菜地、围栏、仓库、净水装置。
- 稳定交易对象。
- 防御设施。
- 小范围救助或雇佣。

变化重点：

```text
我能不能活下来
→ 我能不能活得更稳、更舒服、更安全
```

### 第三阶段：特殊区域探索（81-150章）

功能：避免资源来源只剩开箱。

常用内容：

- 废弃村镇。
- 雾中诊所。
- 旧学校。
- 地下洞穴。
- 污染矿洞。
- 粮仓/水厂/旧车站。

作用：

- 让资源获得有场景感。
- 埋旧世界异常。
- 引出更高级建筑图纸和材料。

### 第四阶段：区域合并与秩序竞争（151-250章）

功能：把矛盾从“人与环境”升级为“人与人/势力/秩序”。

常用内容：

- 多个区域合并。
- 交易价格战。
- 资源产区争夺。
- 投靠强者。
- 安全但被控制 vs 自由但危险。

主角路线：

```text
不主动争霸
但庇护所变强后，被迫承担一部分公共性
```

### 第五阶段：世界规则揭示（250章后）

功能：提供长期追读期待。

常用内容：

- 天灾不是随机。
- 木屋/庇护所不是普通建筑。
- 迷雾在筛选或迁徙玩家。
- 旧世界遗迹和当前求生规则有关。

边界：

- 真相按层揭开。
- 不一次解释完。
- 不让世界真相冲掉每章的资源、升级、交易反馈。

## 四、前期必须轻埋的钩子

前 30 章只轻埋，不展开：

- **废弃村镇**：雾中偶尔出现不属于当前世界的建筑轮廓。
- **区域合并**：交易频道偶尔显示“区域编号”“跨区屏蔽”等字样。
- **木屋异常**：备注出现“成长中”“编号不完整”“尚未绑定最终权限”等模糊提示。

## 五、同题材常见解法

### 个人求生升级为领地/庄园

```text
木屋 → 小院 → 庇护所 → 临时营地 → 小型安全区
```

不要开局写领主争霸。先让庇护所自然长大，再引入少量被庇护者、功能建筑和防守问题。

### 资源问题升级为秩序斗争

中期可以引出“投靠强者”的矛盾：

```text
自由但危险
vs
安全但被控制
```

这个矛盾不需要复杂智斗，但能从单纯开箱升级转向群体冲突。

### 主角爽升级为成果被需要

```text
前期：别人羡慕主角有资源
中期：别人需要主角的交易和庇护
后期：主角的庇护所成为一批人的希望
```

读者追的不只是装备，而是“这个地方会发展成什么”。

## 六、使用边界

- 保下限靠成熟模板，不靠新创复杂体系。
- 中后期升级的是问题层级，不是单纯数值倍率。
- 区域竞争要从生存压力自然产生，不要让反派无脑挑衅。
- 世界真相只负责拉长期待，不要替代每章反馈。
- 新人优先写清楚“缺什么、去哪找、拿到什么、怎么升级、下一章为什么继续看”。
