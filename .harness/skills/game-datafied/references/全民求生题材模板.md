# 全民求生题材边界总纲

> 重参考文件。这里负责回答：`什么算全民求生`、`什么不能写`、`哪些写法是毒点`。
> 如果要设计书名、黄金三章、爽点结构、世界观和阶段发展，去 `projects` 层，不在本文展开。

## 一、总定义

`全民求生` 是全民题材中的一个大类，核心判断标准不是场景，而是玩法。

必须同时满足这四点，才算正统全民求生：

```text
全民投放或全民进入同一套求生规则
+ 开局资源明显稀缺
+ 外部环境持续施压
+ 主角必须靠探索/交换/建设活下去
```

如果少掉其中两项以上，通常就已经偏出全民求生。

## 二、Index：子题材规则入口

这里只做规则导航，不负责写法设计。

| 子题材 | 核心边界文档 | 主要用来判断什么 |
|:------|:-------------|:----------------|
| 迷雾木屋求生 | `全民求生-迷雾木屋求生.md` | 有没有写成雾中探索 + 木屋升级，而不是普通末世囤货 |
| 公路求生 | `全民求生-公路求生.md` | 有没有写出路线、载具、站点代价，而不是旅行流水账 |
| 海岛求生 | `全民求生-海岛求生.md` | 有没有写出海域/岛屿差异，而不是森林换皮 |
| 领地化生存 | `全民求生-领地化生存.md` | 有没有从个人求生自然升级，而不是开局领主争霸 |

## 三、Map：关系边界图

```text
全民求生
├─ 场景差异
│  ├─ 迷雾
│  ├─ 公路
│  └─ 海岛
├─ 成长底盘
│  └─ 木屋/庇护所升级
└─ 中后期升级方向
   └─ 领地化生存
```

边界解释：

- `迷雾`、`公路`、`海岛` 是场景和行动方式，不是另一个大类。
- `木屋/庇护所升级` 是全民求生最常见的成长底盘。
- `领地化生存` 是中后期升级出来的问题层级，不是开局主线。
- 如果一上来就强调领民生产、城池经营、阵营争霸，更接近 `全民领主`。

## 四、全民求生必须保留的底层规则

- 必须有明确资源缺口，如水、食物、燃料、建材、防寒、防守。
- 必须有持续外压，如极寒、黑夜、怪物、雾潮、风暴、区域淘汰。
- 必须有基地成长或生存条件改善，不然容易写成纯逃亡。
- 必须有可见反馈，如升级、交易、开箱、功能解锁、扛过灾害。
- 必须有下一轮压力，不然追读链条会断。

如果正文长期缺少这些内容，就已经在跑偏。

## 五、总毒点

### 5.1 题材跑偏毒点

- 写成普通末世散文，只有气氛，没有规则和反馈。
- 写成囤货日记，长时间没有探索、交换、升级和冲突。
- 写成全民领主换皮，开局就招兵买马、分封管理。
- 写成玄幻缝合，突然加入修仙境界、血脉觉醒、神魔大战。
- 写成全知全能系统文，主角没有生存压力和选择成本。

### 5.2 爽点失效毒点

- 章节里只有找资源，没有明确收获反馈。
- 配角只会无脑嘲讽，打脸方式机械重复。
- 升级只涨数字，不改变生活状态和行动能力。
- 世界频道变聊天室，陌生人高频深聊人生。
- 灾害只当背景板，不真正影响角色决策。

### 5.3 长线崩盘毒点

- 过早解释世界终局，后期失去悬念。
- 中期仍然只靠开箱，资源来源没有场景升级。
- 后期只加数值倍率，问题层级不升级。
- 领地化后没有秩序问题，变成空壳据点。

## 六、金手指边界

全民求生最稳的金手指不是“发资源”，而是“给有限信息差”。

允许：

- 风险提示。
- 物品用途提示。
- 短期灾害预警。
- 低阶材料的隐藏价值提示。

慎用或禁用：

- 无限资源。
- 万倍返还。
- 万能商城。
- 一眼看穿所有秘密。
- 开局直接无敌基地。
- 没有代价的一键升级。

## 七、节奏红线

- 前期必须优先解决“今晚怎么活”这一级问题。
- 中期必须从“活下来”升级到“活得更稳、更舒服、更安全”。
- 后期必须把冲突从个人生存升级为区域、秩序或规则层问题。
- 真相揭示只能拉长期待，不能替代每章反馈。

## 八、自检清单

写大纲或正文前，先过一遍：

```text
这本书的资源缺口是什么？
这本书的持续外压是什么？
这本书的基地成长载体是什么？
这本书每章的可见反馈是什么？
这本书中期的问题层级如何升级？
这本书有没有误写成全民领主/普通末世/玄幻杂糅？
```

如果这六项答不清，就不要继续写正文。

## 九、和 projects 层怎么分工

- `references` 负责：规则边界、禁区、毒点、审稿自检。
- `projects` 负责：书怎么设定、黄金三章怎么开、爽点怎么排、世界观怎么搭、阶段怎么推。

不要把“开书方案”和“规则审稿”写进同一份文档里。
