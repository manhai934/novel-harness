# 游戏题材方向与正文边界

> 重参考文件。仅在生成游戏题材灵感、章节大纲、正文，或审查正文是否跑偏时加载。
> 若已确定使用 `信息提示/迷雾备注`，但需要处理同质化和背景差异化，加载 `提示类金手指差异化.md`。

## 一、方向锁定原则

本参考服务的是**游戏化生存/数据化降临/副本升级**方向，不是泛末世、泛玄幻、泛都市爽文。写作或审稿前必须先锁定本章属于哪一种游戏题材坐标，正文只能围绕该坐标的核心玩法展开。

### 新人保底原则

新人作者、老书虫转作者、脑洞和智斗不稳定时，优先使用**热门成熟模板**，不要一开始新创复杂体系。模板不是上限，但能保住下限：读者至少知道主角在做什么、爽点在哪里、下一章为什么要追。

- 不知道选什么金手指 → 默认推荐 `信息提示/迷雾备注`。
- 不知道怎么推进章节 → 默认使用“缺资源 → 出门探索 → 获得反馈 → 回屋升级/交易 → 章尾新危机”。
- 不知道怎么扩世界观 → 先只写木屋、迷雾、资源、交易、天灾，不写复杂文明史、神明体系、多阵营战争。
- 不知道怎么制造爽点 → 优先用开箱、升级、图纸、交易频道对比、天灾提前准备。
- 不要为了显得新鲜而自创一整套读者不熟的规则。番茄新人文先求清楚、稳定、爽点密，再谈差异化。

## 二、可选方向标签

生成灵感、章节大纲或正文前，先从下列标签中选择 2-4 个作为本章方向：

| 标签 | 适合写什么 | 正文核心爽点 |
|:-----|:-----------|:-------------|
| 游戏降临 | 现实被游戏规则改造，怪物、副本、等级、装备出现 | 世界规则突变、首个优势、别人慌主角稳 |
| 全民转职 | 人人觉醒职业，职业强弱、技能路线、转职任务构成主线 | 废职反转、隐藏职业、首通副本 |
| 全民求生 | 所有人被投放到求生环境，各自开局资源有限 | 开箱收获、信息差、活得比别人好 |
| 庇护所升级 | 小屋、安全屋、木筏、车辆、岛屿等可升级基地 | 从破烂到堡垒的可视化成长 |
| 迷雾/公路/海岛求生 | 固定地图机制：迷雾探索、公路前进、海岛经营 | 探索未知、资源路线选择、区域差异 |
| 副本秘境 | 副本入口、机制、BOSS、首通、掉落 | 机制破解、队伍对比、首通奖励 |
| 天灾轮换 | 高温、暴雨、极寒、虫潮、黑夜等周期灾害 | 提前准备、灾后收割、别人求生主角发育 |
| 信息提示 | 迷雾备注、风险提示、资源提示、用途提示、短期灾害提示 | 靠信息差提前准备，稳定领先但不破坏数值 |
| 交易频道 | 世界频道、区域频道、交易市场、拍卖、私聊 | 信息差变现、资源垄断、公开打脸 |

## 三、推荐组合公式

优先使用明确组合，不要把所有元素一股脑塞进正文。

### 番茄稳妥款

`全民求生 + 庇护所升级 + 天灾轮换 + 交易频道 + 信息提示`

适合写开箱、升级、安全屋变化、灾害倒计时、交易市场打脸。

### 升级爽文款

`游戏降临 + 全民转职 + 副本秘境 + 信息提示 + 排行榜`

适合写废职反转、隐藏机制、首通副本、排行榜震动。

### 探索经营款

`迷雾/海岛求生 + 庇护所升级 + 交易频道 + 区域资源差异`

适合写路线选择、地图探索、资源垄断、区域互补。

### 末日压迫款

`游戏降临 + 天灾轮换 + 副本秘境 + 安全区/庇护所`

适合写规则倒计时、灾害压迫、副本冒险、安全区秩序。

## 四、正文边界

- 正文必须服务当前方向的核心玩法：求生就写资源、庇护所、天气、交易；转职就写职业、技能、副本、排行榜；副本流就写机制、BOSS、掉落、首通。
- 每章至少有一个明确游戏反馈：开箱、升级、掉落、交易成交、公告、技能解锁、庇护所变化、灾害倒计时。
- 金手指要和玩法绑定：提示用于提前准备，备注用于发现用途，合成只服务庇护所/装备升级，不要变成万能许愿机。
- 社交频道只能承担对比、交易、信息差、打脸功能，不能让陌生人无缘无故深聊人生。
- 不要混入无关体系：修仙境界、玄幻血脉、都市神豪、古武宗门、无解释异能、传统丧尸囤货流。
- 不要把求生文写成普通末世散文。没有资源压力、规则倒计时、升级反馈，就不是游戏求生。
- 不要把全民转职写成传统玄幻拜师。职业强弱应来自技能、装备、副本机制，不是天赋血脉。
- 不要把副本秘境写成无限流大杂烩。当前世界的副本必须遵守同一套等级、掉落、时间和进入规则。

## 五、游戏循环引擎

游戏化题材的长线推进靠循环，不靠一次性大脑洞。适合新手使用的保底循环是：

```text
发现缺口/新目标
→ 探索或挑战
→ 获得新资源/装备/图纸/能力
→ 当场形成反馈（升级、交易、打脸、排行榜、解决危机）
→ 引出更高一级的新地图/新怪物/新材料/新灾害
→ 进入下一轮循环
```

不同题材只替换循环里的对象：

| 题材 | 获取什么 | 反馈什么 | 下一轮引出什么 |
|:-----|:---------|:---------|:---------------|
| 全民求生 | 水、食物、木材、图纸、火种 | 木屋升级、交易频道打脸、扛过天灾 | 更远迷雾区、更高级材料、更强灾害 |
| 庇护所升级 | 建材、家具、炉具、防御设施 | 安全度提升、功能房间解锁 | 二级/三级庇护所需求 |
| 副本秘境 | 装备、技能书、首通奖励 | 战力提升、排行榜变化、队友震惊 | 更高难度副本或区域任务 |
| 全民转职 | 技能、职业材料、转职线索 | 废职反转、首通公告、战力排名 | 新职业阶段或隐藏任务 |

### 循环边界

- 每一轮循环必须有明确反馈，不能只探索不收获。
- 新地图/新装备/新灾害要服务当前阶段，不要过早打开世界终局。
- 打脸来自信息差和结果对比，不要让配角无脑嘲讽。
- 升级要有材料成本、时间成本或风险成本，不能一键飞升。
- 循环的目标是稳定追读，不是无限拔高数值。

## 六、金手指边界

新手写全民求生/迷雾庇护所方向，优先使用**信息差型金手指**，不要使用高倍率、无限资源、万能商城等容易导致数值膨胀的金手指。

### 推荐：迷雾备注/提示系统

主角能看到物品、地点、危险和短期事件的简短备注。备注只提供**风险、用途、短期趋势**，不直接提供战斗力，不直接发资源。

```text
【腐朽木箱】
备注：里面有半瓶水和一张破损图纸。箱底有毒蛇，先用木棍敲三下。

【黑曜石碎片】
备注：大多数人会把它当垃圾，但它是二级壁炉的核心材料。

【灰白色雾墙】
备注：六小时后会向内收缩，木屋外的生物会开始迁徙。
```

### 使用限制

- 只提示附近、当前可接触、当前阶段能理解的东西。
- 只提示风险和用途，不直接给完整攻略。
- 可以提示天灾短期趋势，但不能剧透全世界终局。
- 可以帮助主角提前准备，但不能让主角跳过探索、交易、升级和选择成本。
- 高级危险或高级物品可以增加提示消耗、冷却或模糊度。

### 禁用或慎用

- 不用万倍返还、亿倍经验、无限资源、无限合成。
- 不用开局 SSS 级无敌天赋。
- 不用万能商城、签到送神装、抽奖乱出超模道具。
- 不用“一眼看穿所有秘密”的全知提示。
- 不用能直接改写规则的神级系统。

## 七、生成前提示词

写正文前，先在脑内或输出计划中补齐这四项：

```text
本章方向标签：{从方向标签中选择 2-4 个}
核心玩法：{本章主要写求生/转职/副本/庇护所/交易/天灾中的哪一个}
本章反馈：{读者能看到的明确收获或危机变化}
不可越界：{本章不写哪些无关体系或发散内容}
金手指边界：{本章提示只提供什么信息，不能直接解决什么问题}
循环位置：{本章处于缺口/探索/获得/反馈/引出下一轮中的哪一环}
```

如果无法填出这四项，不要直接生成正文，先回到设定或大纲阶段。
