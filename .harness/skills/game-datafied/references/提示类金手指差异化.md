# 提示类金手指差异化

> 重参考文件。仅在用户选择“信息提示/迷雾备注”作为保下限金手指，但担心同质化、想寻找背景差异化时加载。
> 若需要先判断 `全民求生` 下应该走哪条子路线，先加载 `全民求生题材模板.md`。
> 若已经确定是 `迷雾 + 木屋/庇护所` 路线，再回到本文做提示类金手指差异化。

## 一、核心判断

提示类金手指很大众，但它是新人最稳的保底能力。问题不在“提示俗”，而在**背景载体、长期循环和反馈设计太普通**。

正确做法：

- 金手指保持简单：只提供风险、用途、短期趋势，不直接发资源。
- 差异化放在背景故事、庇护所形态、迷雾规则、灾害循环上。
- 不靠万倍、无限、SSS、万能商城制造差异。
- 不追求“别人没写过”，先追求“读者一眼懂、作者每天能写”。

## 二、相近设定类型

| 类型 | 常见设定 | 优点 | 风险 |
|:-----|:---------|:-----|:-----|
| 迷雾提示 | 破草棚/小木屋 + 迷雾 + 资源提示 | 成熟、好理解、长篇验证强 | 同质化最严重 |
| 小木屋提示 | 小木屋升级 + 提示/商店 | 稳，章节动作清楚 | 容易变开箱流水账 |
| 海上提示 | 海洋/木筏/海岛 + 隐藏提示 | 场景差异明显 | 资源体系要重新设计 |
| 弹幕提示 | 能看到局外观众弹幕提醒 | 有新鲜感 | 容易变吐槽流，主线散 |
| 万物心声 | 听到物品/动物/巨物心声 | 轻松、有记忆点 | 对语感和拟人化要求高 |
| 巨物共生 | 在巨鱼/巨兽/活体空间内求生 | 背景强，差异化高 | 生态体系难度高 |

## 三、适合新人使用的背景变体

### 1. 迷雾里的废弃村镇

不是纯森林求生，而是迷雾中周期性出现废弃村镇、旧屋、荒井、诊所、学校、仓库。

优势：熟悉场景变危险，读者容易代入，作者也容易写。

```text
每个人开局一间木屋，木屋外是白雾。
雾中偶尔浮现废弃村镇，里面有水井、粮仓、药店，也有不知道死了多久的“住户”。
```

### 2. 移动安全区

木屋不是永久固定点。迷雾每隔七天收缩，安全区改变，主角需要搬迁、拆建、升级。

优势：天然避免中后期困在小屋里没事写。

```text
木屋升级 → 雾潮逼近 → 探索新区域 → 找搬迁核心 → 建新庇护点
```

### 3. 活体木屋

庇护所不是普通建筑，而是会成长的“木屋胚胎”。主角通过备注知道它需要什么。

优势：庇护所升级有情绪价值，但不必写成复杂万物心声。

```text
【一级木屋】
备注：它很冷，需要火种；它很饿，需要十块干木；它害怕今晚的雾。
```

### 4. 七日轮换灾害

每七天一轮主题灾害：极寒、酸雨、虫潮、黑夜、饥荒、迷雾兽潮、区域合并。

优势：新人最好写，因为每卷都有清楚目标。

```text
第1轮：活过极寒
第2轮：防住虫潮
第3轮：寻找稳定水源
第4轮：区域合并
```

### 5. 物品备注而非系统提示

不要总写“叮！发现宝箱！”，改成物品和地点自身带备注，更自然，也更不容易像传统系统文。

```text
【裂开的水桶】
备注：还能接三次雨水，第四次会漏光。

【发黑的干肉】
备注：吃了不会死，但会拉一整晚。

【旧铁炉】
备注：它不是坏了，只是缺一块黑曜石炉芯。
```

## 四、推荐设定组合

最适合新人、老书虫转作者的组合：

```text
全民迷雾求生
+ 活体木屋/庇护所升级
+ 物品备注型提示
+ 七日一轮天灾
+ 区域频道/交易频道
```

一句话概念：

```text
全民被投放进迷雾世界，每个人开局一间快散架的木屋。
别人只能摸黑找资源，主角却能看到物品和地点的“备注”，一步步把木屋养成能抵御雾灾的庇护所。
```

## 五、不建议新人使用的差异化

- 弹幕提示：容易写成段子，主线散。
- 巨鱼胃中求生：新鲜但生态体系难。
- 克苏鲁深海：氛围强，但设定和文笔要求高。
- 多阵营领主争霸：中后期管理难。
- 复杂规则怪谈：脑洞和逻辑压力大。

## 六、保底原则

老模板打底，小差异包装。

提示类金手指负责保下限，背景变体负责制造记忆点。不要让差异化变成新负担。
