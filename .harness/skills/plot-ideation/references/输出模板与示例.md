# 创作灵感输出模板与示例

> 重参考文件。仅在需要标准化输出报告、示例格式，或用户要求“按模板给我”时加载。

## 一、剧情构思示例

```text
用户：主角刚拿到一条关键线索，接下来怎么写？

方向A：顺着线索追下去，马上撞见第一个阻碍
  - 题材方向标签：悬疑推进 + 线索追踪 + 第一次正面阻碍
  - 冲突：主角想查清真相，但对方已经开始遮掩
  - 核心玩法：查证、试探、碰壁、拿到新信息
  - 爽点类型：信息差爽点 + 破局爽点
  - 篇幅：短期（2-3章）
  - 不可越界：不让主角凭空猜中全部真相，不让反派无脑自爆
  - 风险：推进太快会损失悬念
  - 延续性：中——查到新信息后需要接下一个问题

方向B：先藏住线索，借一次日常冲突试探身边人
  - 题材方向标签：人物关系 + 暗线试探 + 小冲突
  - 冲突：主角不能确定谁可信，只能用小事试探
  - 核心玩法：观察反应、制造误会、筛出可疑对象
  - 爽点类型：智斗爽点 + 关系拉扯
  - 篇幅：中期（5-8章）
  - 不可越界：不写陌生人突然深聊人生，不让主角读心
  - 风险：节奏会比直接追线索慢
  - 延续性：高——能带出人物关系和后续背刺

方向C：线索牵出更大的利益关系，主角被迫站队
  - 题材方向标签：势力冲突 + 利益交换 + 选择代价
  - 冲突：主角掌握的信息对多方都有价值
  - 核心玩法：交换条件、试探底线、被迫选择
  - 爽点类型：选择爽点 + 压迫感
  - 篇幅：长期（10+章）
  - 不可越界：不写反派无脑送死，不写陌生人突然深聊人生
  - 风险：需要设计有脑子的对手
  - 延续性：极高——可以延伸出多条故事线
```

## 二、设定扩展示例

```text
用户：想加一个新组织，但又不想破坏现有格局

推荐方案：外围办事处（只掌握局部资源）

设计定位：
  - 题材方向标签：组织扩展 + 局部冲突 + 信息交换
  - 不直接抬高世界天花板
  - 考验的是人情、信息和资源交换，而不是强行换地图

具体描述：
  - 位置：旧城区一间不起眼的办事处
  - 功能：负责收集消息、调停小冲突、介绍灰色资源
  - 代价：所有帮助都需要人情或把柄交换
  - 边界：只处理局部问题，不掌握终局真相

兼容性：
  - 不改主线权力结构 → 只是补足中间层
  - 帮助有代价 → 不会变成万能工具
  - 可重复使用 → 后续能作为信息入口

故事功能：
  - 给主角一个低成本接触更大世界的入口
  - 制造“帮不帮、信不信、换不换”的选择
  - 后续可引出更高层级势力

正文使用边界：
  - 只用于信息交换和局部冲突
  - 不直接替主角解决主线危机
```

## 三、反转/钩子示例

```text
用户：这章结尾想加个悬念

推荐方案：信息钩子

当前情境：
  - 主角刚从旧档案里找到一个名字
  - 其他人还不知道这条线索

钩子设计：
  “他正要把那张纸塞回文件夹。
  门外有人敲了两下。
  ‘别查这个名字。’”

下一章展开：
  - 敲门的人是谁
  - 对方为什么知道主角查到了这个名字
  - 主角是装傻、反问，还是直接试探
  - 制造“继续查还是先保命”的选择冲突

伏笔回收机会：
  - 如果之前埋过“有人盯着档案室”的线索，这里可以联动回收
```

```text
  - 故事位置：{一句话当前进度}
  - 角色状态：{主要角色当前状态}
  - 可用伏笔：{未回收伏笔列表}

构思方向（共 {N} 个）：

方向A：{标题}
  - 题材方向标签：{例如：悬疑追查 / 关系拉扯 / 求生资源 / 经营扩张}
  - 核心玩法：{本方向主要围绕什么叙事机制展开}
  - 核心冲突：{描述}
  - 爽点类型：{战斗/升级/收获/装逼/情感/成长}
  - 预计篇幅：{短期/中期/长期}
  - 不可越界：{本方向不写什么，避免发散}
  - 风险：{描述}
  - 延续性：{高/中/低}

方向B：{标题}
  ...

对比表：{见上方示例}

建议优先级：{推荐先走哪个方向，为什么}
```
