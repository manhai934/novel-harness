# 创作灵感输出模板与示例

> 重参考文件。仅在需要标准化输出报告、示例格式，或用户要求“按模板给我”时加载。

## 一、剧情构思示例

```text
用户：林北刚拿到双倍奖励，接下来怎么写？

方向A：利用优势快速升级
  - 题材方向标签：游戏降临 + 副本秘境 + 系统倍率
  - 冲突：战斗爽点，靠双倍经验快速冲级
  - 核心玩法：刷副本、拿经验、换装备
  - 爽点类型：升级爽点 + 战斗爽点
  - 篇幅：短期（2-3章）
  - 不可越界：不写修仙顿悟、不写现实神豪、不写无关感情线
  - 风险：容易变无敌流，爽完就没了
  - 延续性：低——升级完了就得找新目标

方向B：低调发育，关键副本一鸣惊人
  - 题材方向标签：全民转职 + 副本秘境 + 交易频道
  - 冲突：蓄力爆发型，先压抑再释放
  - 核心玩法：隐藏收益、准备副本、首通打脸
  - 爽点类型：装逼爽点 + 收获爽点
  - 篇幅：中期（5-8章）
  - 不可越界：不写主角无意义装穷，不写旁人凭空看穿秘密
  - 风险：前期可能太憋屈
  - 延续性：高——一鸣惊人后引发连锁反应

方向C：双倍奖励被发现，卷入阵营冲突
  - 题材方向标签：游戏降临 + 交易频道 + 阵营冲突
  - 冲突：人际冲突 + 资源争夺
  - 核心玩法：资源信息差被放大，引发争夺
  - 爽点类型：情感爽点 + 战斗爽点
  - 篇幅：长期（10+章）
  - 不可越界：不写反派无脑送死，不写陌生人突然深聊人生
  - 风险：需要设计有脑子的对手
  - 延续性：极高——可以延伸出多条故事线
```

## 二、设定扩展示例

```text
用户：想加一个新副本，但又不想破坏数值平衡

推荐方案：镜像副本（怪物复制玩家属性）

设计定位：
  - 题材方向标签：副本秘境 + 系统倍率
  - 不需要固定数值，任何等级都能打
  - 考验的是战斗技巧而非硬属性

具体描述：
  - 入口：每个主城都有一个镜像大厅
  - 机制：副本怪物精确复制玩家的等级和属性
  - 奖励：通关后根据“效率分”发放（越快越好）
  - 特殊掉落：镜像碎片（可兑换技能点）

兼容性：
  - 不依赖固定数值 → 任何等级都能打
  - 奖励与投入匹配 → 不会造成数值膨胀
  - 持续可重复 → 是日常副本而非一次性的

故事功能：
  - 让主角展示战斗技巧（别人打不过是因为菜）
  - 可以设置排行榜 → 装逼爽点
  - 技能点兑换 → 长期成长线

正文使用边界：
  - 只用于副本机制、排行榜和技能点成长
  - 不引出修仙心魔、血脉觉醒、现实神豪等无关体系
```

## 三、反转/钩子示例

```text
用户：这章结尾想加个悬念

推荐方案：信息钩子

当前情境：
  - 林北刚打完副本，在清点收获
  - 队伍其他人在安全区等他

钩子设计：
  “林北正要退出副本界面——屏幕上突然弹出一条系统消息。
  他看了一眼，手指僵住了。”

下一章展开：
  - 消息内容：一个限时紧急任务
  - 任务奖励无法拒绝
  - 但必须独自前往，不能带队友
  - 制造“去还是不去”的选择冲突

伏笔回收机会：
  - 如果之前埋过“系统任务有隐藏条件”的线索，这里可以联动回收
```

```text
方向B：{标题}
  ...

```
