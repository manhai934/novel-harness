# 句式节奏档案

> 6 类句式技巧，让叙述有呼吸感、有变化、有"人味"。

---

### 4.1 长短句交替

AI的问题不是"句子太长"或"句子太短"——而是**没有变化**。

```
AI式（均匀长度，每句10-15字）：
  他从楼道里走了出来。外面的天色已经暗了。走廊的灯光亮了起来。他看了一眼手里的纸条。

人写（长短交替）：
  他推门出去。天已经暗了。走廊的灯亮了起来。（短）
  他低头瞄了一眼手里的纸条。（中）
  还差一点。（极短，强调）
```

**门禁**：相邻三句的长度不要相近。长-短-长，或短-短-长，就是不要"均匀"。

---

### 4.2 断句与留白

AI不喜欢断句——它倾向于把话说完整。人写东西的时候会：

- 说到一半断开（因为不知道怎么说）
- 用破折号插入另一个念头
- 突然换话题
- 话说一半被别的东西打断

```
AI式：他想告诉她真相，但话到嘴边又咽了回去，因为他知道现在还不是时候。
人写：他张了张嘴——算了。还不是时候。
```

**反门禁：断句不是把结论拆成两句解释。**

```text
❌ 不是水里的味道。是他嘴里。
❌ 不是风声。是脚步声。
❌ 不是害怕，而是身体先反应过来了。
```

这些句子看起来短，实际是作者替读者盖章。遇到这种结构，改成感官和动作：

```text
✅ 腥味从喉咙底下翻上来。他抿了下嘴，舌根发苦。
✅ 风停了一下。门外那点声响还在，一下一下，贴着木板往前蹭。
```

---

### 4.3 语气随情绪变化

叙述的语气应该跟着角色的状态走，而不是一直保持"标准叙事腔"。

```
平静时：句子完整，用词正常
紧张时：句子变短，动词前置，不用修饰
烦躁时：用词变粗，语气词增多，句子碎片化
高兴时：语气上扬，用词变轻，句子节奏变快
```

**门禁**：连续读三页正文，看看叙述语气有没有变化。从头到尾一个调 → 违规。

---

### 4.4 "把"字句的语感

"把"字句是中文口语的标志性结构。AI用得少，人用得多。

```
AI式：他推开了门。                 人写：他把门给推开了。
AI式：她撕下一张报纸。             人写：她把手里的报纸给撕下来一张。
AI式：他锁上了门。                 人写：他把门给锁上了。
```

**使用时机**：
- 日常动作 → 优选"把"字句（喝茶、开门、拿东西）
- 紧急动作 → 不用"把"字句（冲过去、闪开、躲）
- 正式叙述 → 看节奏，交替使用

---

### 4.5 重复强调的节奏

AI从不重复，人靠重复制造节奏。不可滥用——只在关键场景用一次，就有重量。

**三种重复模式：**

**模式一：动词重叠（日常自然）**
```
他点了点头。→ 他点点头。
她笑了笑。  → 她笑了笑。（"笑了笑"比"微笑了一下"自然十倍）
他试了试。  → 他试了试。
```

**模式二：同一词/短语句尾回响（情绪节点）**
```
沉默。沉默。
黑暗。无边无际的黑暗。
他什么也没说。一句话也没说。
```

**模式三：形容词重叠强化（环境/感受）**
```
黑乎乎的 → 黑乎乎的一片
静悄悄的 → 静悄悄的一点声音都没有
亮堂堂的 → 亮堂堂的跟白天一样
```

**门禁**：找到一段情绪最强烈的场景，看有没有用重复强调。没有 → 考虑加一个。

**限制**：重复强调只能重复感受、动作、环境词，不允许重复判断结论。不要用 `不是X。是Y。` 伪装成强调。

---

### 4.6 心理代偿短语

中文口语最独特的人味标志——把简短的内心活动"摊开"成更长的内心独白。

| AI简洁版 | 人写代偿版 | 效果 |
|:---------|:-----------|:------|
| 告诉自己那是老鼠 | 在心底里头跟自己说那准是老鼠 | 拉长 + 口语化 |
| 他知道她在笑 | 可他心里头清楚得很，她一定在笑 | 转折 + 肯定 |
| 他认为 | 他还觉得 / 他心里头不这么认为了 | 软化判断 |
| 他决定先去主卧 | 就决定先去二楼主卧那边把行李给整理一下 | 啰嗦但真实 |
| 他想了想 | 他在心里头琢磨了一阵子 | 拉长时间感 |
| 她有点紧张 | 她心里头七上八下的，也不知道在慌个什么劲 | 展开 + 自问 |
| 他有点犹豫 | 他拿不定主意，去了怕尴尬，不去又不甘心 | 正反拉扯 |

**使用时机**：一场戏中用 1-2 次就够了。用在角色面临抉择或情绪波动的时刻。通篇都用 → 啰嗦过头。
