# 语感对照词典

> 不是"对错"问题，而是"AI更倾向于用A，人更倾向于用B"的统计差异。
> 8 张对照表，覆盖动词、副词、连词、人称、数量、动词扩展、虚词语气词、套话替换。

---

### 3.1 动词选择

| 场景 | AI倾向 | 普通人倾向 |
|:-----|:-------|:-----------|
| 开始做某事 | 他开始/他开始进行/他着手 | 他就/他直接/他干脆 |
| 做决定 | 他决定/他选择/他下定决心 | 他想/他打算/他干脆/他就 |
| 思考 | 他思考/他思索/他陷入沉思 | 他想了想/他琢磨/他盘算 |
| 看到 | 他看见/他注意到/他观察到 | 他看到/他瞄见/他瞥见 |
| 说话 | 他说道/他开口/他发言 | 他说/他开口/他来了句 |
| 走路 | 他迈步/他步行/他前行 | 他走/他往那边去/他过去 |
| 使用 | 他使用/他运用/他采用 | 他用/他拿起/他抄起 |
| 结束 | 他结束/他完成/他终止 | 他搞完/他弄好/他就这么 |

### 3.2 副词选择

| AI高频 | 人话替代 |
|:--------|:----------|
| 迅速地/快速地 | 直接 / 一下子 / 几下 |
| 缓慢地/慢慢地 | 慢慢 / 一点一点 / 磨蹭 |
| 仔细地/认真地 | 仔细 / 认真（去掉"地"） |
| 轻轻地/小心地 | 轻 / 小心（去掉"地"） |
| 愤怒地/生气地 | 直接写动作，不加副词 |
| 惊讶地 | 直接写动作，不加副词 |

**门禁**：中文的"地"大部分时候可以去掉。"他迅速地站了起来" → "他一下子就站起来了"。

### 3.3 连词选择

| AI高频 | 人话替代 |

| 然而/但是 | 但 / 直接接 |
| 因此/因而 | 所以 / 于是 / 直接接 |

### 3.4 人称指代

| 场景 | AI倾向 | 普通人倾向 |
|:-----|:-------|:-----------|
| 首次引入 | 这个叫做王磊的男人/一名叫李思雨的女性 | 王磊/李思雨 |
| 重复提及 | 我们的主角林北/这位玩家 | 林北/他 |
| 多人场景 | 后者/前者/此人 | 直接写名字/他 |
| 身份指代 | 作为队长的王磊/身为医生的李思雨 | 王磊/李思雨 |

### 3.6 动词扩展（口语化替换）

AI偏好简洁的二字动词，人偏好更具体、更长、更有画面感的表达。

| AI简洁版 | 人写扩展版 | 说明 |
|:---------|:-----------|:------|
| 寻找 | 四处寻摸 / 东翻西找地寻摸 | 加了动作过程 |
| 观察 | 仔细打量了一番 | "仔细"+"一番"拉长节奏 |
| 休息 | 歇上一歇 / 找个地方歇一会儿 | 口语化，有停顿感 |
| 确认 | 反复确认了好几遍 | "反复"+"好几遍"强调次数 |
| 等待 | 干等着 / 百无聊赖地等着 | 加了情绪修饰 |
| 逃跑 | 撒开腿就跑 / 拔腿就往外窜 | 画面感 |
| 思考 | 在心里头盘算了一阵 | 心理代偿 |
| 整理 | 里里外外拾掇了一通 | 口语词"拾掇" |
| 打扫 | 上上下下收拾了一遍 | "上上下下" = 全方位的口语版 |
| 检查 | 里外看了个遍 / 翻来覆去查看 | 动作过程展开 |
| 说话 | 开了口 / 来了句 / 蹦出一句 | 比"说"更生动 |
| 吃饭 | 扒拉了几口 / 凑合吃了一顿 | 口语化 |
| 睡觉 | 倒头就睡 / 眯了一会儿 | 具体姿态 |
| 喝水 | 灌了一口 / 抿了一口 | 具体方式 |
| 走路 | 溜达 / 晃悠 / 快步走 | 不同节奏对应不同心情 |

### 3.7 虚词与语气词对照

人写东西时靠虚词堆出"口气"，AI用得太少。

| 手法 | 用法 | 示例 |
|:-----|:-----|:------|
| 句末"了" | 补时态感，几乎句句可加 | "已经发黄" → "已经发黄了" |
| "就" | 增加顺承感 | "他到了门口" → "到了门口就走了进去" |
| "都" | 加强语气 | "他没想到" → "他都没想过这事" |
| "倒" | 轻微转折/语气软化 | "还行" → "倒也还行" |
| "还" | 延续/补充 | "他记得那天" → "他还记得那天的事" |
| "也" | 并列/追加 | "他没说话" → "他也没说话" |
| "可" | 轻微转折 | "但他没去" → "可他没去" |
| "罢了" | 句末语气 | "运气好罢了""刚好碰上罢了" |
| "呢" | 句末缓和 | "还早着呢""急什么呢" |
| "嘛" | 句末理所当然 | "本来就这个价嘛""算了嘛" |
| "那个""这个" | 指示词增加口语感 | "老妇人" → "那个老妇人" |
| "下来""起来""出来" | 补语词拉长动作 | "亮了" → "亮了起来""坐" → "坐了下来" |
| "有点""差不多""左右" | 程度模糊化 | "几秒" → "有几秒钟""十个人" → "差不多十个人" |

**门禁**：连续 5 句话没有一个"了/就/都/倒/还/也/可" → 缺口气，违规。

### 3.8 套话替换速查表

AI高频套话 → 人话替代，按优先级排列。

| 优先级 | AI原词 | 人话替代 |
|:-------|:-------|:---------|
| ★★★ | 值得注意的是 | 注意 / 要提醒的是 / 特别说一下 |
| ★★★ | 综上所述 | 总之 / 说到底 / 简单讲 |
| ★★★ | 与此同时 | 同时 / 这时候 / 另一边 |
| ★★★ | 由此可见 | 所以 / 可见 / 看得出 |
| ★★★ | 显而易见 | 明摆着 / 很明显 |
| ★★☆ | 值得一提的是 | 值得说的是 / 顺带一提 |
| ★★☆ | 简而言之 | 简单说 / 一句话 |
| ★★☆ | 此时此刻 | 眼下 / 这个时候 / 此刻 |
| ★★☆ | 不仅仅是 | 不光是 / 不只是 / 不止是 |
| ★★☆ | 近年来 | 这些年 / 这几年 |
| ★☆☆ | 小心翼翼 | 仔细 / 谨慎 / 留神 |
| ★☆☆ | 就在这时 | 这时 / 就这当口 / 正这时候 |
| ★☆☆ | 逐渐 | 渐渐 / 慢慢 / 一点点 |
| ★☆☆ | 突然 | 猛地 / 冷不丁 / 一下子 |
| ★☆☆ | 非常 | 很 / 特别 / 格外 |

**门禁**：一篇 3000 字的章节，套话出现超过 3 次 → 必须替换。
