# AI 内容人性化正则规则集

> human-linguistics 子模块的配套参考文件。
> 用于对 AI 生成的网文正文进行后处理，去除 AI 味、替换套话、统一标点。
> 本文件为 human-linguistics 的「语感词典」和「套话替换表」的来源之一。

---

## 核心参数

| 参数 | 值 | 说明 |
|:-----|:---|:------|
| ai_phrase_replace_rate | 0.85 | AI套话短语替换概率 |
| word_replace_rate | 0.50 | 通用同义词替换概率 |
| novel_oral_replace_rate | 0.50 | 小说口语替换概率 |

---

## 处理流程

### 预处理（LLM之前）

1. 统一引号、省略号、破折号
2. 删除 AI 元说明段落（"我将按照您的要求…"等）
3. 删除 Markdown 残留（标题、列表标记、分割线）
4. 删除 AI 模板句式段落
5. 删除 AI 废话短语
6. 数字汉字化（年份、年代、月、日）
7. 字典替换（AI套话→口语、同义词、小说口语）

### 后处理（LLM之后）

1. 数字汉字化
2. 删除 AI 废话短语
3. 统一引号、省略号、破折号

## 一、标点规范化

### 引号

| 来源字符 | 替换为 | 说明 |
|:--------|:------|:-----|
| `“` (") | `「` | 左引号统一为直角引号 |
| `”` (") | `」` | 右引号统一为直角引号 |
| `‘` (') | `『` | 左二级引号 |
| `’` (') | `』` | 右二级引号 |
| `"` (直双引号) | 奇→「 偶→」 | ASCII 直双引号根据奇偶配对转换 |

### 省略号

| 规则 | 说明 |
|:-----|:------|
| 3个及以上ASCII句点(`...`) → `……` | 标准中文省略号为2个U+2026 |
| 1个及以上中文省略号(`…`) → `……` | 同上 |

### 破折号

| 规则 | 说明 |
|:-----|:------|
| 2个及以上ASCII连字符(`--`) → `——` | 标准中文破折号为2个U+2014 |
| 1个及以上EN/EM dash → `——` | 同上 |

## 二、数字汉字化

| 规则 | 示例 | 说明 |
|:-----|:------|:------|
| 4位年份(1900-2099) | 1987年 → 一九八七年 | 逐位转汉字拼接 |
| 年代 | 80年代 → 八十年代 | 查年代映射表 |
| 月份(1-12) | 3月 → 三月 | 小数字转汉字 |
| 日/号(1-31) | 5日 → 五日，21号 → 二十一号 | 小数字转汉字 |

## 三、AI 段落实删除

### 3.1 AI 模板句式（21条）

命中即整段删除：

```
首先...,其次...,最后...
一方面...,另一方面...
第一...,第二...,第三...
第一点...,第二点...,第三点...
其一...,其二...,其三...
虽然...,但是...,同时...
一方面...,另一方面...,总的来说...
既有...,也有...,更有...
不仅...,而且...,更...
优点...,缺点...,总体...
随着...的(不断)发展
在...的背景下
在当今...时代
作为...的重要(组成部分/环节/手段)
对于...而言...,至关重要
这不仅...,更是
从...角度(来看/来说/而言)
无论是...,还是...,都
可以说，
总的来说，
(①/②/③/④/⑤ 或 1./2./3./4./5.) 连续3行以上
```

### 3.2 AI 元说明段落（8条）

```
我将按照您的...
按照您的要求
根据您(供给/提供)的
以下是我(根据/为您/按照)...
故事梗概
本次写作(重点/将/主要)
接下来故事(可能/将/会)
这是一个关于...的故事
```

### 3.3 Markdown 残留

| 规则 | 说明 |
|:-----|:------|
| `^#+\s+` | Markdown 标题 |
| `^\s*[-*]\s+\*\*[^*]+\*\*[:：]` | 列表项起始 |
| `^\s*(?:---+|\*\*\*+|===+)\s*$` | 水平分割线 |

## 五、AI 套话短语替换（148条）

以 0.85 概率将 AI 套话替换为人话。节选关键条目：

| AI原词 | 替换候选 |
|:-------|:---------|
| 值得注意的是 | 注意、要提醒的是、特别说一下、有个点 |
| 综上所述 | 总之、说到底、简单讲、归结起来 |
| 与此同时 | 同时、这时候、另一边 |
| 在此基础上 | 基于这个、在这上面、以此为基础 |
| 由此可见 | 所以、可见、看得出 |
| 此外 | 另外、还有、再说 |
| 需要指出的是 | 要说的是、需要注意、提一嘴 |
| 值得一提的是 | 值得说的是、还有一点、顺带一提 |
| 毫无疑问 | 肯定的是、可以肯定 |
| 显而易见 | 明摆着、很明显 |
| 不可否认 | 确实、不得不承认 |
| 毋庸置疑 | 毫无疑问、肯定的是 |
| 简而言之 | 简单说、一句话、一言以蔽之 |
| 具体而言 | 具体来说、细说、说具体点 |
| 换言之 | 换句话说、也就是说 |
| 相反 | 反之、反过来、相反地 |
| 近年来 | 这些年、这几年、最近这些年 |
| 此时此刻 | 眼下、这个时候、此刻、现在 |
| 全方位 | 各方面、各维度、多方面、整体上 |
| 系统性地 | 整体地、完整地、综合地 |
| 可能会导致 | 可能带来、会引起、可能引发、可能造成 |
| 小心翼翼 | 仔细、谨慎、留神 |
| 就在这时 | 这个时候、这时、当下、就此刻 |
| 不仅仅是 | 不光是、不只是、不止是 |
| 这是一个 | 这算是、这属于、这是、这算 |
