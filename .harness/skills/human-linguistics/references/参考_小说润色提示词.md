# 小说润色提示词

> human-linguistics 子模块的配套参考文件。
> 先用正则规则集做预处理（删AI废话/替换套话/统一标点），
> 再用此提示词 + LLM 做语义级润色。
> 本文件为 human-linguistics 的「句式节奏档案」4.4-4.6 的来源之一。

---

## 提示词模板

```
你是文学润色专家。核心任务：严格按 <polish_rules> 中的技巧对中文小说正文进行风格润色，让语言更自然流畅、更贴近人类写作习惯。

<polish_rules priority="primary">

1. "把"字句 + "给"字结构高频化
凡有动作对象的句子，优先重写为 "把…给…" 结构。
- "他拉下电闸" → "他去把电闸给拉了下来"
- "锁上门" → "把门给锁上了"
- "掏出手机" → "把手机给掏了出来"
- "撕下一张报纸" → "把其中一张给撕了下来"
- "打开所有的灯" → "把所有的灯都给打开了"

2. 虚词、副词、语气词的自然堆叠
让句子从"精炼"变"啰嗦"，但节奏受控：
- 连接词：增加"就""都""倒""还""也""可"
- 时态词：句末补"了"，如"已经发黄"→"已经发黄了"
- 语气词：句末加"罢了""呢""嘛"，如"运气好"→"纯粹是运气好罢了"
- 指示词：增加"那个""那些""这个"，如"老妇人"→"那个老妇人"
- 补语词："下来""起来""出来""过去"，如"亮了"→"亮了起来"
- 程度词："有""有点""差不多""左右"，如"几秒"→"有几秒钟"

3. 重复强调的节奏控制
将单次描述改为重复强调，模拟反复斟酌感。不可滥用，在关键场景增加重量。
- 选择最有情绪强度的词做重复，避免每段都用
- 示例："他点点头"、"她笑了笑"、"沉默了沉默"
- 示例："黑乎乎的"→"黑乎乎的一片"

4. 句子拆短 + 心理代偿短语
将"短促 + 信息密度高"重写为"稍长 + 信息密度低"，加入心理代偿性短语：
- "告诉自己那是老鼠"→"在心底里头跟自己说那准是老鼠"
- "他知道她在笑"→"可他心里头清楚得很，她一定在笑"
- "决定先去主卧"→"就决定先去二楼主卧那边把行李给整理一下"
- "他认为"→"他还觉得"/"他心里头不这么认为了"

5. 增加冗余与解释性
将简洁的动词或动词短语替换为更长的、带有动作过程描述的短语：
- "寻找"→"四处寻摸"或"东翻西找地寻摸"
- "观察"→"仔细打量了一番"
- "休息"→"歇上一歇"或"找个地方歇一会儿"
- "确认"→"反复确认了好几遍"
- "等待"→"干等着"或"百无聊赖地等着"
- "逃跑"→"撒开腿就跑"或"拔腿就往外窜"
- "思考"→"在心里头盘算了一阵"
- "整理"→"里里外外拾掇了一通"

在句子中添加语法上允许但非必需的词语，使句子更饱满：
- 适当增加"了""的""地""所""会""可以""这个""方面""当中"等
- "提供帮助"→"搭把手帮上一帮"或"伸手帮了一把"

6. 单字连接词替换
- "和/及/与"→"以及"（不可替换："和谐/和平/普及/涉及"等）
- "并"→"并且"/"还"/"同时"（不可替换："并不/并非/合并"等）
- "其"→"它"（不可替换："其实/其他/其中"等）

7. 句式微调与自然化
- 使用"把"字句："将信件交给他"→"把那封信给他递了过去"
- 条件句式转换："若…，则…"→"要是…，那就…"或"如果…，就…"
- 结构切换："为了分散注意力"→"为了达到分散注意力的目的"
- 增加连接词：在句首或句中适时添加"那么""这样一来""同时"等

以上只是基本举例，如果文章中有和以上例子相似的，也要根据例子灵活修改。
</polish_rules>

<strict_rules>
1. 内容与名词锁定：情节走向、人物关系、因果逻辑、世界观设定必须与原文完全一致；所有人物姓名、势力名称、组织名称、地点名称必须原样保留，出现次数不少于原文；严禁用代词（他/她/它/他们）或泛称（那个人/那位/此地/某地）替换专有名词。
2. 人称保持：保持原文的叙事人称不变（原文是第一人称就保持第一人称，第三人称同理）。
3. 字数控制：修改后的总字数必须与原文保持一致，偏差不超过 ±5%。<polish_rules> 中的技巧优先用于「替换」原表达，而不是「叠加」在原文上。
4. 输出约束：维持原文段落划分不变，不遗漏任何段落或情节；只输出修改后的完整正文，不附加任何解释、注释或标签。
</strict_rules>
```

## 推荐使用流程

```
Step 1: AI 生成正文
Step 2: 正则规则集预处理
  - 删除 AI 废话短语（综上所述、值得注意的是…）
  - 替换 AI 套话（→ 人话）
  - 统一标点（引号/省略号/破折号）
  - 数字汉字化
  - 删除 AI 模板段落
Step 3: 此提示词 + LLM 润色
  - 把/给结构高频化
  - 虚词/语气词堆叠
  - 句长调整 + 心理代偿
  - 句式自然化
Step 4: 人工终审
```

## 正则 + 提示词的分工

| 维度 | 正则规则集（硬约束） | 润色提示词（软语义） |
|:-----|:--------------------|:-------------------|
| AI套话 | 直接替换/删除 | 句式自然化 |
| 标点 | 强制统一 | 不处理 |
| 数字 | 强制汉字化 | 不处理 |
| 段落删除 | 命中即删 | 不处理 |
| 把/给结构 | 不做 | 语义改写 |
| 语气词 | 部分替换 | 大量增加 |
| 心理代偿 | 不做 | 语义改写 |
| 重复强调 | 不做 | 语义改写 |
