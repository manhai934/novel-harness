# 网文写作基础包

## 定位

本包提供通用网文写作方法，服务开书、黄金三章、章节规划、正文生成和审稿。

## 黄金三章目标

第一章：

- 快速给出主角处境。
- 抛出核心异常或核心压力。
- 让读者知道这本书的卖点是什么。

第二章：

- 让主角做第一次选择。
- 展示金手指、能力、资源或关系的第一轮作用。
- 建立短期目标。

第三章：

- 给出第一次可见成果或第一次失败代价。
- 扩大冲突，让读者知道下一阶段要看什么。

## 章节基础结构

```text
承接上章 → 当前问题 → 角色行动 → 阻碍升级 → 结果反馈 → 章尾钩子
```

## 爽点不是只有赢

常见爽点：

- 信息差：读者知道主角掌握了别人没有的线索。
- 选择爽：主角在压力下做出有效选择。
- 收获爽：资源、关系、能力、地位发生变化。
- 反制爽：前文压力在当前章节被局部反打。
- 认知爽：读者对规则、人物或世界有新的理解。

## RAG 使用边界

适合：

- 开书规划。
- 章节写前准备。
- 章尾钩子设计。
- 大纲可行性检查。

不适合：

- 替代具体题材包。
- 在没有项目上下文时直接生成长篇正文。
