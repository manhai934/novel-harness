# 去 AI 味基础包

## 定位

本包提供通用去 AI 味参考，用于 RAG 在审稿、润色、正文生成自检时补充规则。

## 高危指纹

- 解释腔：连续使用“因为、所以、这意味着、换句话说”。
- 判定式短句：`不是X，是Y`、`不是X。是Y。`、`不是X，而是Y`。
- 自问自答：角色刚遇到问题，旁白立刻替他分析完整答案。
- 总结腔：段尾用“这一刻，他终于明白……”收束。
- 机械因果：每个动作都被解释成明确目的。

## 修正方向

- 用动作、气味、触感、物件状态替代解释。
- 保留一点无效观察和口气，不让每句话都服务信息推进。
- 紧张场景可以短，但不能全是断言式短句。
- 对话不要过度承担设定说明，允许误判、打岔和废话。

## RAG 使用边界

适合命中：

- 用户说“这段太 AI 了”。
- 审稿 Agent 查解释腔、总结腔、自问自答。
- 写作 Agent 完成初稿后做轻量自然化。

不适合：

- 替代题材专项规则。
- 把所有题材都改成同一种粗糙口气。
