## 约束 1：成长曲线 — 非数值驱动

```
核心规则：
  □ 属性只在关键节点提升（转职/大阶段突破），不随等级自动增长
  □ 战斗胜负由机制决定，不是数值碾压
  □ 每次副本的"收获"以新机制/新知识/新资源为主
  □ 成长体现在"能做以前做不到的事"，而非数字变大
```

## 约束 2：数值膨胀控制 — 严格

```
属性增长：仅在大节点触发，每次需剧情铺垫
装备系统：改造制（禁止替换制，打到蓝装不扔白装）
副本产出：图纸/材料/信息碎片/功能道具（禁止三件套）
天花板机制：每个大阶段设绝对属性上限，达到后只能靠新机制突破
```

## 约束 3：同质化防控 — 强制

```
Phase 1 (1-15章): 生存适应期 — 求生+建设安全屋
Phase 2 (16-40章): 副本发育期 — 刷副本+资源收集
Phase 3 (41-70章): 势力接触期 — 玩家交易/合作/竞争
Phase 4 (71-120章): 世界观揭秘期 — 探索末世真相
```

## 约束 4：长久发展 — 阶段天花板

```
每个大阶段结束时软重启：属性达到天花板 → 解锁下一阶段新机制 → 旧的数值部分归零或转换 → 用新机制继续成长。
```

| 行为 | 判定 |
|:-----|:------|

| 瓶颈+突破 | ✅ 鼓励 |
