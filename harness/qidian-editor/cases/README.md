# 案例库（Cases）

> 每次审稿/创作过程中发现的典型案例，结构化归档。
> 目标是积累足够多的正反案例，让 Agent 能从过往经验中学习。

---

## 目录结构

```
cases/
├── README.md              ← 本文件，案例库说明
├── 语病案例/               ← 从 human-linguistics 实战对照库衍生
│   ├── 2026-05-06-第1则-多动症叙述.md
│   ├── 2026-05-06-第2则-解释狂.md
│   └── ...
└── 审稿案例/               ← 每次审稿的结构化记录
    └── template.md
```

## 案例分类

| 类型 | 内容 | 来源 |
|:-----|:-----|:------|
| 语病案例 | 具体的 AI 语病 → 人写改法 | human-linguistics 实战对照库、审稿中发现 |
| 审稿案例 | 完整审稿过程 + 发现 + 修正 | 每次审稿任务 |
| 创作案例 | 剧情构思过程 + 选择理由 | 创作灵感任务 |

## 使用方法

1. 发现典型案例 → 按模板创建案例文件
2. 同类错误出现 3 次 → 考虑升级为高频规则
3. 定期回顾，合并重复，淘汰过时案例
