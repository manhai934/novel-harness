# 角色关系金字塔与对话边界

> 迁移自主模块 3.8。属于 plot-review 维度一：角色行为一致性的核心判定依据。

---

## 核心问题

AI最常见的社交逻辑错误：把初次见面的陌生人写成认识多年的老友。不同关系层级的人，对话边界完全不同，而且**对对方性格/行为模式的了解程度也完全不同**。

---

## 完整关系金字塔

```
                     至交/密友（数月~数年）
                  ╱  知根知底，无话不谈  ╲
                 ╱  能预判对方行为/性格  ╲
                ─────────────────────────
                     朋友（数周~数月）
                  ╱  了解性格习惯偏好  ╲
                 ╱  可坦诚交流多数话题  ╲
                ─────────────────────────
                    熟人/固定队友（一周以上）
                  ╱  仅知表面行为特征  ╲
                 ╱  知道"这人话少/话多"  ╲
                ─────────────────────────
                    陌生人/初次合作（刚认识）
                  ╱  完全不了解对方  ╲
                 ╱  仅从外表/动作看表面  ╲
                ─────────────────────────
```

---

## 各层级对对方的了解程度

| 层级 | 了解程度 | 能预测什么 | 不能预测什么 |
|:-----|:---------|:-----------|:-------------|
| **Lv.1 陌生人** | 零了解 | ❌ 无法预测任何人的行为模式 | "王磊迟早会看到"——你怎么知道王磊细心？ |
| | | ❌ 无法判断谁是什么性格 | "李思雨迟早会算"——你怎么知道她爱算账？ |
| | | ✅ 只能基于系统规则做推断 | "等级公开，明天谁都看得到"——规则推断 |
| **Lv.2 熟人** | 知道表面行为特征 | ✅ "赵凯话多""王磊话少" | ❌ 不知道对方的核心动机 |
| **Lv.3 朋友** | 了解性格习惯 | ✅ "王磊这人较真" | ❌ 核心秘密 |
| **Lv.4 至交** | 知根知底 | ✅ 能预判多数行为 | — 无话不谈 |

---

## 两条核心写作原则

**原则一：沉默是正常的**——不是每次对视都需要内心独白。

**原则二：不知道写什么的时候，什么都不写。**

---

## 两种不同性质的"判断"

```
系统规则推断（✅ 允许）
  → "等级在副本里公开，明天谁都看得到"
  → 不管对方是什么人，事实就是如此

性格行为预判（❌ 刚认识时不允许）
  → "王磊迟早会看到他的等级"
  → 你怎么知道王磊是细心的人？

强行解读眼神/表情（❌ 无话找话）
  → "那一眼的意思林北懂——..."
  → 对方可能根本没想那么多
```

---

## 特殊例外：高智商/天赋型人物的有限推测

以上规则适用于普通人。但如果角色明确设定了超高智商/极强观察力/特殊洞察类技能，在合理范围内可以有少量额外推断。但必须：

- 天赋在设定中明确写了（不能临时加）
- 推断基于可观察的细节线索（行为/装备/语言习惯）
- 推断是试探性的（"可能……"），不是笃定的结论
- 仍然不能看透人心（高智商≠读心术）
