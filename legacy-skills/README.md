# 旧版 Skill 资产

这个目录保存的是早期从其他项目参考、整理或导入的 skill 包。

它们暂时保留，主要作为后续迁移素材使用，价值集中在：

- 市场扫榜和热门题材总结
- 长篇/短篇写作流程
- 去 AI 味规则
- 拆文和分析模板
- 浏览器/CDP 辅助脚本

当前项目的真实入口是根目录：

- `README.md`：给用户看的说明入口
- `SKILL.md`：给总编 Agent 看的调度入口

不要把本目录当成当前对外暴露的主入口。旧版 skill 中稳定、可复用的流程规则，后续应迁移到 `SKILL.md` 或 `.harness/agents/`；可复用的参考资料、题材模板、案例和提示词，后续应迁移到 RAG 参考库。

保留这个目录的目的不是继续扩展旧架构，而是给后续合并提供一个清晰的资产暂存区。
